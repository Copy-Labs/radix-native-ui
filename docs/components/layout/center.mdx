# Center

`Center` is a utility component for centering content horizontally, vertically, or both. It also provides convenience wrappers (`HCenter`, `VCenter`, `AbsoluteCenter`) for common use cases.

## Import

```tsx
import { Center, HCenter, VCenter, AbsoluteCenter } from '@radix-ui/themes-native';
```

## Usage

### Center Both Axes (default)

```tsx
import { Center, Text } from '@radix-ui/themes-native';

<Center style={{ height: 200 }}>
  <Text>Centered content</Text>
</Center>
```

### Center Horizontally Only

```tsx
<Center axis="horizontal">
  <Text>Horizontally centered</Text>
</Center>
```

### Center Vertically Only

```tsx
<Center axis="vertical" style={{ height: 200 }}>
  <Text>Vertically centered</Text>
</Center>
```

### Using HCenter and VCenter

Convenience wrappers for common centering:

```tsx
// HCenter - Horizontal centering only
<HCenter>
  <Text>Horizontally centered</Text>
</HCenter>

// VCenter - Vertical centering only
<VCenter style={{ height: 200 }}>
  <Text>Vertically centered</Text>
</VCenter>
```

### AbsoluteCenter

For centering with absolute positioning:

```tsx
<Box style={{ position: 'relative', width: 200, height: 200 }}>
  <AbsoluteCenter>
    <Spinner />
  </AbsoluteCenter>
</Box>
```

## Axis Values

| Axis | Description | CSS Equivalent |
|------|-------------|----------------|
| `both` | Center horizontally and vertically | `align-items: center; justify-content: center` |
| `horizontal` | Center horizontally only | `align-items: center` |
| `vertical` | Center vertically only | `justify-content: center` |

## Radius Values

| Radius | Description |
|--------|-------------|
| `none` | No border radius |
| `small` | Small border radius |
| `medium` | Medium border radius |
| `large` | Large border radius |
| `full` | Fully rounded (circle) |

## When to Use

- **Center**: When you need to center content in a flex container
- **HCenter**: When you only need horizontal centering
- **VCenter**: When you only need vertical centering
- **AbsoluteCenter**: When you need to center content using absolute positioning (overlays, positioned elements)
