# Kbd

`Kbd` is a typography component for rendering keyboard shortcuts and key combinations.

## Import

```tsx
import { Kbd } from 'radix-native-ui';
```

## Usage

```tsx
import { Text, Kbd } from 'radix-native-ui';

<Text>
  Press <Kbd>Cmd + C</Kbd> to copy.
</Text>
```

## Props

<PropTable>
  <Prop name="children" type="ReactNode">
    Key content.
  </Prop>
  <Prop name="style" type="StyleProp<TextStyle>">
    Custom styles.
  </Prop>
  <Prop name="color" type="'white' | 'black' | 'gray' | 'gold' | 'bronze' | 'brown' | 'yellow' | 'amber' | 'orange' | 'tomato' | 'red' | 'ruby' | 'crimson' | 'pink' | 'plum' | 'purple' | 'violet' | 'iris' | 'indigo' | 'blue' | 'cyan' | 'teal' | 'jade' | 'green' | 'grass' | 'lime' | 'mint' | 'sky'">
    Text color.
  </Prop>
  <Prop name="backgroundColor" type="string">
    Background color.
  </Prop>
  <Prop name="size" type="1 | 2 | 3 | 4 | 5 | 6 | 7 | 8 | 9" default="1">
    Size of the keyboard key.
  </Prop>
</PropTable>

## Examples

### Single Key

```tsx
<Text>
  Press <Kbd>Enter</Kbd> to submit.
</Text>
```

### Key Combinations

```tsx
<Text>
  <Kbd>Cmd + S</Kbd> to save
</Text>

<Text>
  <Kbd>Ctrl + Shift + P</Kbd> to open command palette
</Text>
```

### Navigation Keys

```tsx
<Flex direction="column" gap={2}>
  <Text><Kbd>↑</Kbd> Move up</Text>
  <Text><Kbd>↓</Kbd> Move down</Text>
  <Text><Kbd>←</Kbd> Move left</Text>
  <Text><Kbd>→</Kbd> Move right</Text>
</Flex>
```
