# Strong

`Strong` is a typography component for rendering bold text for emphasis.

## Import

```tsx
import { Strong } from 'radix-native-ui';
```

## Usage

```tsx
import { Text, Strong } from 'radix-native-ui';

<Text>
  This is <Strong>important</Strong> text.
</Text>
```

## Props

<PropTable>
  <Prop name="children" type="ReactNode">
    Text content.
  </Prop>
  <Prop name="style" type="StyleProp<TextStyle>">

  <Prop name="color" type="'white' | 'black' | 'gray' | 'gold' | 'bronze' | 'brown' | 'yellow' | 'amber' | 'orange' | 'tomato' | 'red' | 'ruby' | 'crimson' | 'pink' | 'plum' | 'purple' | 'violet' | 'iris' | 'indigo' | 'blue' | 'cyan' | 'teal' | 'jade' | 'green' | 'grass' | 'lime' | 'mint' | 'sky'">
    Text color.
  </Prop>
</PropTable>

## Examples

### Inline Emphasis

```tsx
<Text>
  This is <Strong>bold</Strong> text inline.
</Text>
```

### With Color

```tsx
<Text>
  Warning: <Strong color="red">This action cannot be undone.</Strong>
</Text>
```

### In Heading

```tsx
<Heading size={4}>
  Welcome to <Strong>Radix Themes Native</Strong>
</Heading>
```

### Multiple Emphasis

```tsx
<Text>
  The <Strong>quick</Strong> brown fox jumps over the <Strong>lazy</Strong> dog.
</Text>
```
