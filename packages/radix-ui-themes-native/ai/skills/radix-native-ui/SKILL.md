# Copy Native UI Skill

Skill Name:
copy-native-ui

Purpose:
Enable AI agents to generate React Native and Expo user interfaces using Copy Native UI primitives.

Framework:
radix-native-ui

Author:
Copy Labs

Capabilities:
- Install Copy Native UI
- Generate components
- Compose UI primitives
- Scaffold mobile screens
- Follow mobile-first UI patterns

Platforms Supported:
- Expo
- React Native

Component Philosophy:
Copy Native UI follows a composable primitive architecture inspired by Radix UI and shadcn-style component registries.

Usage Examples:

Use skill: copy-native-ui

Generate a login screen using Copy Native UI.

Generate a profile settings screen using Copy Native UI components.
