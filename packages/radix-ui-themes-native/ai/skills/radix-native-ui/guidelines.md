# UI Generation Guidelines

When generating UI with Copy Native UI:

1. Always import components from "radix-native-ui"
2. Prefer primitives over raw React Native components
3. Follow composable UI patterns
4. Ensure mobile-safe layouts
5. Use consistent spacing
6. Keep UI accessible
7. Prefer functional React components

Do NOT:

- mix multiple UI frameworks
- create unnecessary wrappers
- duplicate primitives
